MEETILY THAI - BUILD TROUBLESHOOTING (WINDOWS)

ERROR 13: Git for Windows is missing
  winget install --id Git.Git -e --source winget

ERROR 14: Rust/Cargo is missing
  winget install --id Rustlang.Rustup -e --source winget

ERROR 17: CMake is missing
  winget install --id Kitware.CMake -e --source winget

After an installation finishes:
1. Close the current builder window.
2. Open a new Command Prompt.
3. Run BUILD_WINDOWS_INSTALLER.cmd again.

The r4 builder searches standard installation folders and Visual Studio's
bundled CMake before offering automatic installation through WinGet.

Recommended project folder: C:\MeetilyThai
